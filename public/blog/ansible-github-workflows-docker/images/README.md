# Images Directory

Place all images for this blog post in this directory.

## Banner Image

The banner image should be named `banner.jpg` (or update the `bannerImage` field in `post.json` to match your filename).

## Getting the Banner from Medium

To download the banner image from your Medium article:

1. Open the Medium article in your browser
2. Right-click on the banner/header image
3. Select "Save image as..." or "Copy image"
4. Save it as `banner.jpg` in this directory

Alternatively, you can use browser developer tools:
1. Open the article and press F12
2. Find the image element in the HTML
3. Copy the image URL
4. Download it using a tool like `curl` or your browser
